# Frontend
## 개발 환경
- OS: Windwos
- 언어: TypeScript
- 프레임워크: Next.js 15, React 19

## 기술 스택
- CSS: Tailwind CSS
- UI: shadcn/ui
- 상태관리: Zustand
- 폼: Reach Hook Form + Zod

## 추가 사항
- any 타입 사용 금지
- 컴포넌트 분리 및 재사용
- 반응형 필수

# Backend
## 아키텍처
- 레이어드 아키텍처 (Controller -> Service -> Repository)
- DTO 패턴 사용
- 의존성 주입

## 추가 사항
- 에러 핸들링 필수
- DB 트랜잭션 처리
- API 응답 형식 일관성 유지
